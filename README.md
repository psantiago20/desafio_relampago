## Sobre o projeto

Este projeto consiste no desenvolvimento de um site de portfólio pessoal, criado como parte de um desafio prático da disciplina de UI/UX e Front-end.

O objetivo principal é aplicar, de forma organizada e progressiva, os conceitos de:

- HTML semântico  
- Organização visual com CSS  
- Interações básicas com JavaScript  
- Boas práticas de versionamento com Git e GitHub  

O site é estruturado em painéis (Home, Sobre, Trabalhos e Contato), com navegação fixa e comportamento de rolagem controlado.

---

## Objetivos da aula

- Criar páginas utilizando HTML semântico  
- Separar corretamente HTML, CSS e JavaScript  
- Desenvolver layout responsivo  
- Aplicar princípios básicos de UI/UX

## Versionamento e fluxo de trabalho

O projeto segue um fluxo de versionamento organizado, conforme boas práticas.

Branches utilizadas:

main  
Contém a versão final e estável do projeto  

develop  
Branch principal de desenvolvimento  

feature/html-structure  
Criação e finalização do HTML  

feature/css-layout  
Criação e finalização do CSS  

feature/js-interactions  
Criação e finalização das interações em JavaScript  

Commits seguem o padrão:

feat: add semantic HTML structure  
style: finalize CSS layout and responsiveness  
feat: add navigation and scroll interactions  

Cada etapa foi desenvolvida, commitada e integrada separadamente.
